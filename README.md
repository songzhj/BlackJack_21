# BlackJack_21
## 实现思路
为了实现多人联机，使用Node做游戏的服务器，与前端的通信使用socket.io来实现WebSocket。

## 使用说明
在项目目录下执行`npm install`安装所需要的依赖。服务器地址默认使用的localhost本机IP，可自行修改。启动服务器后可以访问client/index.html创建房间，复制生成的链接邀请好友加入游戏。
